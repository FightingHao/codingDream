今日，上海中原地产公开回应称，客观市场环境的确恶劣，但从我国整体国民经济形势来看，仍然是稳中向好，韧性十足，所以，中原的伙伴们依然要对市场和行业抱有期待，在目前这个阶段更要做好自己，传递自己的正能量。 上海中原总经理陆成表示“有同事看到个别媒体报道上海中原包括减佣金和减福利报酬等节流措施，问我应对的意见。其实，现在连龙头开发商万科都高呼要‘活下去！’我倒觉得中原人实话实说更好，这并不丢人。眼前的市场环境实在是太恶劣，调控政策严厉，市场成交量锐减，金融去杠杆持续，行业产能过剩都是事实，我们只是主动采取相对温和的节流措施应对漫长寒冬，这些降低营运成本的不得已措施旨在尽可能避免大规模裁员关店行动而已。如果我们在公司亏损的情况下仍然‘巍然不动’，什么都不做，那才是对员工和股东不负责任。” 上海中原地产称，为了应对整个市场的变化，上海中原在尊重员工价值的前提下做出了“津贴”和中后台“佣金”的调整，这并不会影响到业务部门的分佣结构和比例的变化，属于公司合理的调整范围。而在《通告》中也指明了所有的调整只是暂时性的，待市场回暖，公司扭亏为盈后，将会做重新评估。
